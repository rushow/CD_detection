# CD Detection

This repository contains the source code and resources for a continuous research project focused on **Concept Drift (CD)**. The goal of this project is to develop and refine methods for detecting concept drift in various datasets over time.

## Project Overview

- **Type**: Research Project
- **Status**: Ongoing
- **Focus**: Concept Drift (CD)

## Folder Structure

- `myenv/`: Local virtual environment (ignored in version control).
- `.idea/`: IDE-specific settings (ignored in version control).

## How to Contribute

As this is an ongoing research project, contributions are welcome. Please ensure that any changes align with the project's goals and are well-documented.
